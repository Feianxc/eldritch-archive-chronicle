# PRODUCT

## Name
不可名状作品编年史

## Purpose
A complete static Lovecraftian/Cthulhu-inspired chronology website. It presents a curated sequence of weird-fiction works, publication history, title aliases, full-text links and relationship networks through a premium occult museum interface.

## Audience
Readers, collectors, students of weird fiction, and design-minded literature fans who want a moody but readable chronology.

## Register
brand/editorial

## Pages / sections
- Cinematic first-viewport archive cover
- Featured manuscript card
- Data-driven works library
- Chronology deep timeline
- Mythos atlas with node interactions
- Work relationship network with relation filters
- Research/search desk
- Literature source and rights boundary note
- Open-source project note

## Current product scope
- Static HTML/CSS/JS site
- `data/works.json` with 35 work records, 6 source groups and 33 curated relationship edges
- Publication venue/issue, collaborator/credit notes, full-text links and title aliases for work records
- Local search, era filters, topic filters, random archive selection, atlas node linking and relationship network filtering
- Generated hero, atlas, reading desk, relationship observatory and work-thumbnail assets
- GitHub Pages deployment
- Vercel production deployment and custom Cloudflare subdomain deployment

## Design constraints
- Dark abyssal editorial style
- Clear readable Chinese interface
- Timeline plus relationship network as the main information architecture
- Real image assets for texture, background, manuscript, atlas, network and thumbnails
- No gore and no cheap horror props
- Responsive desktop and mobile
- Static, deployable to GitHub Pages, Vercel and custom DNS subdomains
